##  Descripción breve
El repositorio SignBridge es un proyecto de desarrollo web construido con **Django** (framework de Python).  
Su estructura y archivos indican que es una aplicación web que combina **backend en Python/Django** con **frontend en HTML, CSS y JavaScript**.

---

## Qué hace y cómo funciona
- **Backend (Django):**
  - Usa `manage.py` para ejecutar comandos del servidor y gestionar la aplicación.
  - Tiene una base de datos `db.sqlite3` para almacenar información.
  - El archivo `requirements.txt` define las librerías necesarias para que el proyecto funcione (probablemente Django y dependencias relacionadas).
- **Frontend:**
  - Carpeta `templates/core` → contiene las páginas HTML que se muestran al usuario.
  - Carpeta `static` → guarda estilos CSS, scripts JS y recursos como imágenes.
- **Ejecución típica:**
  1. Instalar dependencias con `pip install -r requirements.txt`.
  2. Ejecutar el servidor con `python manage.py runserver`.
  3. Acceder a la aplicación desde el navegador en `http://127.0.0.1:8000`.
